# Constellations

This is a game about finding order in chaos. The original concept was to find a set of shapes which are known among a sea of stars. This led to an initial experiment of a set of 6-10 regular polygons (triangle, square, pentagon) at different scales and rotations as the base game. Making all the shapes irregular causes it to be incredibly difficult even at fairly low shape counts. Shapes were shown in a bank, stars selected, and if a correct combination was found then the shape would get filled in and the stars disabled, plus the shape in the bank lit up.

We're interested in various ways that the patterns can emerge for the player. This could include levels which are only rectangles (not rotated, parallel to the screen edges), or having the star sizes proportional to the scale of the shape (with some random noise), or make all of the stars in a given shape twinkle together, or make the segments between various stars that are or aren't connected sometimes fade into and out of existence. Also instead of just doing closed shapes, constellations could be a network of points that isn't a filled in region (imagine a triangle of points which aren't connected to each other but to their centroid). Another angle would be reusing stars in multiple constellations and finding minimal point sets which encode a larger set of interesting shapes.
